<html>
Контейнер для всего содержимого веб-страницы.

<html> Контент </html>

<head>
Предназначен для хранения элементов, цель которых — помочь браузеру в работе с данными (закрывающий тег не обязателен). Содержимое напрямую не отображается на веб-странице, за исключением тега <title>.

<head> Заголовок страницы и техническая информация </head>

<meta>
Определяет метатеги, которые используются для хранения информации предназначенной для браузеров и поисковых систем. Разрешается использовать более чем один метатег (закрывающий тег не требуется).

<meta Описание сайта, ключевые слова и т.д.>

<title>
Определяет заголовок документа для браузера. Необходимо размещать в контейнере <head>.

<title>Wanna sleep!</title>

<body>
Хранит содержание веб-страницы, отображаемое в окне браузера.

<body> Контент </body>

<font>
Контейнер для изменения характеристик шрифта, таких как размер, цвет и гарнитура(устаревший).

Большой красный текст

<p><font size=5 color=FF0000>Большой красный текст</font>

<center>
Выравнивает содержимое контейнера по центру.

Надо маме показать!

<p><center>Надо маме показать!</center></p>

<br>
Устанавливает перевод строки в том месте, где этот тег находится (закрывающий тег не требуется).

вод

  1. Это
  2. нумерованный
  3. список

  • список

<table>

<tr>
